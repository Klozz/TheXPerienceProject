The XPerience Project

XPerience is a custom ROM based on AOSP (Android Open Source Project), with optimizations in system and kernel, to create a great combination between power and good battery performance. Because we care that you can use your device for a long time without resorting to lowering the power of it..

  • Substratum Supported.
  • Upgradable via OTA.
  • Configurable.

Substratum

Themeable

OMS is a client for managing overlays, allowing the providers to dynamically control priorities and enable/disable overlays..

Updates via XPerience Center

When a new compilation is released, you will know with Xperience™ Center, it will display a notification message with the new update available.

Configurable

The XPerience ROM have a lot of features configurables, and inside have Alessa™ kernel tweaker to configure kernel options.
